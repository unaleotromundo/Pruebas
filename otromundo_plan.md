# Landing Page OtroMundo - Plan de Desarrollo

## Objetivo
Crear una landing page para OtroMundo que permita a almacenes del barrio presentar sus servicios con promociones, fotos con precios, comentarios de clientes y chat en vivo.

## Estructura de la Página

### 1. Header/Navegación
- Logo de OtroMundo
- Menú de navegación (Inicio, Servicios, Testimonios, Contacto)
- Botón de "Registrarse" destacado

### 2. Hero Section
- Título principal: "Lleva tu almacén al siguiente nivel con OtroMundo"
- Subtítulo explicativo del servicio
- Call-to-action principal
- Imagen hero atractiva

### 3. Sección de Servicios
- Promociones destacadas
- Galería de fotos con precios
- Sistema de gestión de contenido simple
- Herramientas de marketing digital

### 4. Galería de Ejemplos
- Fotos de productos con precios superpuestos
- Diferentes categorías de productos
- Diseño tipo grid responsivo

### 5. Testimonios de Clientes
- Comentarios de almacenes que ya usan el servicio
- Fotos de los dueños/encargados
- Calificaciones con estrellas

### 6. Chat en Vivo
- Widget de chat flotante
- Registro simple por email
- Filtrado básico de usuarios

### 7. Formulario de Registro
- Campos: Nombre del almacén, Email, Teléfono, Dirección
- Validación de email
- Términos y condiciones

### 8. Footer
- Información de contacto de OtroMundo
- Enlaces a redes sociales
- Información legal

## Tecnologías a Utilizar
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Backend: Flask (Python)
- Base de datos: SQLite para simplicidad
- Chat: WebSocket con Socket.IO
- Estilos: CSS Grid, Flexbox, animaciones CSS

## Paleta de Colores
- Primario: #2563eb (azul profesional)
- Secundario: #10b981 (verde éxito)
- Acento: #f59e0b (naranja llamativo)
- Neutros: #1f2937, #6b7280, #f9fafb

## Funcionalidades Clave
1. Registro de usuarios por email
2. Chat en vivo con moderación básica
3. Galería interactiva de productos
4. Sistema de testimonios
5. Formularios de contacto
6. Diseño completamente responsivo
